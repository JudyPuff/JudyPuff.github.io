Discussing with a non-tech friend:

  1. What a responsive site is, and why responsiveness is important.

    2. A responsive site is a website that reacts to the screen size. Not all screens have the same pixels (pixels are little lights that light up in different ways to make the words and images that you can see on your screen) and so a web designer will have to design their website on different canvases. This is important because nowadays people use different size screens and you want the same information on your website to be displayed on all screens.

  2. What mobile first design is, and why it's important.

    3. Mobile first is design is when web developers design a website to be seen on a mobile/smartphone screen. In the past, the websites are made, and are viewed on desktop computers. Desktop computers have mostly similar sized screens. However, nowadays more people are using their mobile phones to access the internet and so web designers respond to the trend by designing a website to be used on the mobile phone first.

  3. What frameworks are, and their pros and cons.

    4. Frameworks are bits of code that almost all designers (of a similar website) need to use to create a skeleton website. This makes the process of making a website much faster and allows web designers to go straight to designing the website rather than spend a long time setting up the skeleton.
