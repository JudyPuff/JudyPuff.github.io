Technical Blog 8 - Scope

Explain to a non-tech friend what 'scope' is, and how it works in JavaScript.

Before we understand 'scope' we need to understand what a variable is. A variable in JavaScript (and maybe other languages) is a tag or a name for a thing. In real life you have a thing (or a person) THEN you give it a name (Hi! My name is Judy!). In programming, you give something a name then you create the thing (aka data). You hold the data in the variable and then you write code to describe what that data is.

Scope is just the type of variable it is. Local variables can change from function to funciton. Just like in English 'stomach' is that midriff area of the body and in Maori it is 'puku'. It's describing the same thing but the name changes. You can set a 'global scope' so that you can't change the name and whenever you mention the word within the whole document, the computer will know what you mean.
