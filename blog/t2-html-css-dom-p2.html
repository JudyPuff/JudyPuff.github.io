Technical Blog 2 (Reflections on CSS)

What is grid based design?

Designing a website based on a grid or boxes layout.

Whats all the hype about responsive webpages?

Responsive design is becoming more and more relevant because people don't just look at screens (with similar pixels) on a desktop computer any more. There are different resolutions (different pixels) on different devices such as a smartphone or tablet. Therefore, the web pages you design must be "responsive" to the device the user is using.

What is semantic structure?

A "tag" (usually with < > brackets around it) that describes the part of the webpage. For example: means you are linking a folder to this page - like links to my other blogs at the top of this page!

Any other points you found interesting and want to reflect on.
